5.47 Extract Phrases


Extract Phrases can be used to extract only the text that has been tagged. The tags appear in SGML format and can be alphabetically organized (Sort Phrases). This tool is useful when checking for mistakes in tagging or analyzing the tags alone.

Click on the Select Source button to choose the text from which the tool is to extract the tagged phrases.

Click on the Select Phrase Types button to choose the tag types to extract. All the tag types that appear in the document are offered as viewing options. The tag types are selected by clicking on the little box to the left of the type name. The tag types can appear in various colors within the viewer by selecting the white box that appears to the right of the chosen tag type and choosing a color.

Sort Phrases sorts the phrases according to their tags which are themselves sorted alphabetically. If Sort Phrases is left inactive the tagged phrases will be produced according to the order they appear within the text document.

Extract Phrases runs the extraction program.
Click on the Save As button to save the list of extracted phrases as a text document.
