5.5 The Help Menu

Under the Help menu, the following options is available:

Show Mouse Bindings:
Launches a window that displays mouse bindings to be used in tagging. More description of the Mouse Bindings can also be found in chapter 9.Tagging Files for Entities

About: Loading Documents
Provides information to the user about loading raw and sgml-encoded texts.

About:Using the Segmentation Editor
Provides information to the user about the Segmentation Tagger.
