5.45 Learn Alembic Phrase Finding Rules

Using the Rule Learner
New: name the new set of rules
The Training Sources are documents that have been tagged and from which the computer learner will create reading and tagging rules.
The Testing Sources are documents upon which the Learner will tests its rules.
To add a training or testing file simply press the add button and select a file.
To remove a training or testing file, select the file, then press the remove button.

Once the training and testing files have been entered press the Start Learning button to start the rule learning process.
