5.31 Auto tag is a simple automatic tagging mechanism. The user has a choice of four modes, and should set the mode according to his preferences for aided tagging.

Off mode:
Offers no assistance to the user. This is the Alembic Workbench's default mode.

Query mode:
Queries the user to determine whether to apply tag to each subsequent occurrence of a phrase previously tagged. User decides whether to simply add an annotation to the phrase or replace existing annotations.

Auto mode:
Globally adds tags to all occurrences of phrase tagged by user. No feedback is given. To selectively override auto mode when tagging a phrase, press the key sequence Shift-mouse-button 2.

Auto-Confirm mode:
Informs user whether subsequent occurrences of phrase have been found and confirms whether to tag all other instances globally.
